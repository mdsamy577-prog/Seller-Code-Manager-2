==================================================
GLOBAL DISCOUNT SETTINGS
==================================================

Create a new section:

Settings → Global Discount

Title:
Global Discount

Dropdown Options:

- No Discount (0%)
- 20% Discount
- 30% Discount
- 40% Discount
- 50% Discount

Save Button:
Save Discount Settings

==================================================
FUNCTIONALITY
==================================================

When the admin selects a discount percentage and saves:

Apply that discount automatically to ALL package prices across the entire website.

This includes:

- Registration page
- Package cards
- Pricing sections
- Upgrade/Renewal pages
- Any place where package prices are displayed

==================================================
ADMIN CONTROL
==================================================

Only one global discount can be active at a time.

Changing the percentage instantly updates all displayed prices.

Selecting "No Discount (0%)" removes all discount badges and shows normal prices.
